1.

C#

Voto: 9

C# è un linguaggio di programmazione ideato da Microsoft nel 2000.
Influenzato fortemente da Java e C++, C# si propone di semplificare e potenziare le regole dei linguaggi di programmazione precedenti in un linguaggio multi-paradigma fedele però quasi interamente alla sintassi base del C.
Il linguaggio nasce per sfruttare a pieno le potenzialità del framework di Microsoft ".NET".
C# è un linguaggio multi piattaforma, permette di programmare applicativi per dispositivi Android e iOS grazie al framework Xamarin, oppure dispositivi desktop tramite Mono. Il game engine Unity utilizza come linguaggio principale C# per i suoi script.

Curiosità: il suo nome è pronunciato 'see sharp' che letteralmente significa 'vedere nitidamente': uno degli obiettivi del linguaggio.

Potente

2.

Python

Voto: 8

Python è un linguaggio di programmazione ideato da Guido van Rossum all'inizio degli anni novanta.
Come C# è un linguaggio multi-paradigma e multi piattaforma, con la particolarità di essere molto semplice da imparare e immediato nella comprensione e manutenzione del codice.
Il suo punto di forza è la sua comunity attiva, la quale pubblica e cura i pacchetti e le librerie che rendono ampiamente utilizzato il linguaggio. Su PyPI (Python Package Index) è possibile trovare delle librerie aggiornate per ogni scopo: numpy per implementare calcoli matematici algebrici e molto altro, pytorch per sfruttare i modelli sul machine learning.

Curiosità: scordatevi il ';' perché basta andare a capo! Fate attenzione però alle indentazioni: esse servono per aprire un blocco di codice, un po' come '{'

Veloce

3.

Java

Voto: 8

Java è un linguaggio di programmazione ideato da un gruppo di sviluppatori della società Sun Microsystems capitanati da James Gosling nel 1995.
Java è storia! L'idea di base dietro il linguaggio è quella di mantenere la sintassi del C++, linguaggio ampiamente diffuso nel 1995, ma eliminando caratteristiche ritenute non necessarie per il linguaggio e fonte solamente di bug nei programmi.
La sintassi a oggetti di Java e le sue regole di denominazione delle classi hanno fatto la storia dell'informatica.
Le app di Android sono nativamente sviluppate principalmente in Java, anche se ultimamente a quest'ultimo è preferito Kotlin, un linguaggio più veloce e moderno.
Dopo AMOS BASIC è stato il mio primo linguaggio di programmazione ed entra a pieno diritto in classifica.

Curiosità: il primo nome del linguaggio era 'Oak': una quercia che Gosling osservava spesso dalla finestra del suo ufficio, ma esisteva un linguaggio di programmazione già con quel nome e dunque si optò per il nome Java.

Un must

4.

Javascript

Voto: 7 ½

Javascript è un linguaggio di programmazione ideato da Brendan Eich nella seconda metà degli anni novanta in solamente 10 giorni.
Javascript nasce come linguaggio per il web lato client, ma grazie al suo successo e alla sua semplicità ha conquistato milioni di programmatori in tutto il mondo.
Negli anni sono stati sviluppati vari framework per evolvere il linguaggio e permettere la programmazione lato server con Node.js oppure sviluppare app native con React.
Al giorno d'oggi Javascript è il linguaggio di programmazione più utilizzato al mondo.

Curiosità: il nome Javascript è stato scelto solo per motivazione commerciale: Java era già famosa ai tempi.

Onnipresente

5.

PHP

Voto: 7

PHP è un linguaggio di programmazione ideato da Rasmus Lerdorf nel 1994.
PHP è il linguaggio con cui sono stati scritti buona parte dei server dei siti web più importanti per la storia del primo Internet. Proprio per tale motivo i programmatori di PHP sono ricercati e il linguaggio è aggiornato per risolvere problemi di sicurezza.
PHP è un linguaggio di scripting interpretato. Esistono moltissimi framework di terze parti basati su PHP che permettono di incrementare la produttività, la velocità e la robustezza.

Curiosità: Wordpress è un content management system scritto in PHP. Dunque PHP è implicato in più pagine web di quanto non si pensi.

Elefantesco

0.

AMOS BASIC

Voto: ∞

AMOS BASIC è un linguaggio di programmazione ideato da François Lionet nel 1990.
Come bonus AMOS avrà sempre un posto nella classifica dei miei linguaggi preferiti perché ho iniziato il mio percorso nella programmazione con la sua sintassi passando i pomeriggi a divertirmi ragionando.
Il linguaggio è sviluppato per i computer Commodore Amiga. Deriva dallo STOS BASIC, ideato nel 1988, scritto per i computer Atari ST dallo stesso Lionet.

Curiosità: Lo stesso François Lionet, oltre ad aver ideato il linguaggio, ha programmato da solo l'intero IDE.
